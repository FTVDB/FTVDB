# FTVDB

FTVDB is an independent public update metadata reference for Amazon device firmware and system component updates, powered by user submissions and maintained for public research.
Submissions are processed, validated, and added to a public metadata database.

> This project does not host, mirror, modify, proxy, or distribute firmware files, system component packages, or application files.
> FTVDB does not promote jailbreaking, firmware downgrades, bypassing device security, or unauthorized modifications.
> Rights holders can contact support@ftvdb.com for review.

## Official Website

Visit [ftvdb.com](https://ftvdb.com) to:

- Browse firmware metadata for supported device families
- Browse system component metadata
- View public update metadata records
- Submit public, unauthenticated source URLs for review

## How It Works

1. Community submissions report publicly visible firmware and system component source URLs from user-owned devices.
2. The service validates submitted URLs and stores metadata in JSON files.
3. The public database makes reviewed metadata available for research, repair context, and historical reference.

## Repository Contents

- `/database` stores firmware and system component metadata in JSON format.
